#### HR Data Analytics Project
> Comprehensive analysis of HR data to improve workforce management and support strategic decision-making.

- Collected and cleaned employee data using **Python (Pandas, NumPy)** to ensure high-quality datasets for accurate analysis.  
- Conducted **Exploratory Data Analysis (EDA)** to identify trends and patterns in employee information such as age, gender, experience, tenure, and job level.  
- Developed **interactive dashboards using Power BI** to monitor key HR metrics like employee retention, absenteeism, and performance indicators.  
- Provided **actionable insights** to help HR optimize workforce planning, training, and talent management strategies.  
- Highlighted **relationships between factors** such as years of experience and employee satisfaction to guide motivation and development initiatives.
